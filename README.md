# Proyecto-SEAE
Sistema de Evaluación de Alternativas Económicas (SEAE)
## Descripción
Backend del Sistema de Evaluación de Alternativas Económicas (SEAE),  
desarrollado con Spring Boot. Permite evaluar alternativas de inversión  
mediante los métodos: 
-VPN
-CAE 
-TIR.

---

## Tecnologías
- Java 17
- Spring Boot 4.0.6
- Maven
---
## Dependencias

- Spring Web
- Spring Validation
- Lombok

---
## Ejecucion
mvn spring-boot:run
---

##Servidor
http://localhost:8080
